# Shipping Management System (Console App)

Ένα σύστημα διαχείρισης φορτίων πλοίων υλοποιημένο σε C# (.NET). Η εφαρμογή προσομοιώνει τη φόρτωση εμπορευματοκιβωτίων (containers) σε πλοία και υπολογίζει το κόστος μεταφοράς βάσει συγκεκριμένων τύπων.

> **Note:** Το project αποτελεί προσαρμογή πανεπιστημιακής άσκησης (αρχικά για Java/Swing) σε C# Console Application, εστιάζοντας στις αρχές του Αντικειμενοστραφούς Προγραμματισμού (OOP).

## 📋 Περιγραφή

[cite_start]Το σύστημα διαχειρίζεται δύο κατηγορίες εμπορευματοκιβωτίων[cite: 1]:

1.  [cite_start]**Ξηρού Φορτίου (Bulk/Dry):** Χαρακτηρίζονται από το βάρος τους[cite: 3].
2.  [cite_start]**Ψυγεία (Refrigerator):** Χαρακτηρίζονται από την ισχύ που καταναλώνουν[cite: 3].

Ο χρήστης μπορεί να διαχειριστεί πολλαπλά πλοία, να προσθέσει containers (ελέγχοντας τη χωρητικότητα) και να υπολογίσει το συνολικό κόστος ναύλου.

## 🚀 Λειτουργίες

- [cite_start]**Διαχείριση Πλοίων:** Δημιουργία και διαχείριση λίστας πλοίων με συγκεκριμένη χωρητικότητα (capacity)[cite: 5, 12].
- **Φόρτωση Container:** \* Επιλογή τύπου (Bulk ή Refrigerator) μέσω μενού.
  - [cite_start]Εισαγωγή κωδικού, προορισμού και τεχνικών χαρακτηριστικών[cite: 2].
  - [cite_start]Έλεγχος χωρητικότητας πλοίου πριν την προσθήκη (Validation)[cite: 8].
- **Οικονομικοί Υπολογισμοί:**
  - Υπολογισμός κόστους ανά container βάσει μαθηματικού τύπου.
  - [cite_start]Υπολογισμός συνολικού κόστους φορτίου για κάθε πλοίο[cite: 8].

## 🏗️ Αρχιτεκτονική (Structure)

Το project ακολουθεί καθαρή αρχιτεκτονική με διαχωρισμό της λογικής (Domain) από το UI:

- **Domain/**: Περιέχει την επιχειρησιακή λογική.
  - [cite_start]`Container.cs`: Abstract κλάση βάσης με κωδικό και προορισμό[cite: 2].
  - `BulkContainer.cs`: Κληρονομεί από Container (υλοποιεί τη λογική βάρους).
  - `RefrigeratorContainer.cs`: Κληρονομεί από Container (υλοποιεί τη λογική ισχύος).
  - `Ship.cs`: Διαχειρίζεται τη λίστα των containers και τους κανόνες χωρητικότητας.
- **Program.cs**: Η εφαρμογή κονσόλας που προσομοιώνει τις οθόνες διαχείρισης (Menu, User Input).

## 📐 Κανόνες Υπολογισμού

Οι χρεώσεις υπολογίζονται με τους εξής τύπους, όπως ορίζονται στην εκφώνηση:

[cite_start]**Για Container Ξηρού Φορτίου[cite: 4]:**

$$
Χρέωση = 10 \times Βάρος (kg)
$$

[cite_start]**Για Container Ψυγείο[cite: 4]:**

$$
Χρέωση = 2000 \times Ισχύς (kW)
$$
